MOTools Documentation
=====================

MOTools is a Python library for managing machine learning model training and evaluation pipelines.

Features
--------

* **Training Backend Abstraction**: Unified interface for different training backends (OpenAI, custom implementations)
* **Evaluation Backend Support**: Integrated evaluation using Inspect and other frameworks
* **Content-Addressed Caching**: Efficient caching system for training and evaluation results
* **Dataset Management**: Flexible dataset loaders and processors
* **Settings Zoo**: Pre-configured settings for common ML tasks
